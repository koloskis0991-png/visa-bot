# Запуск бота на Railway (бесплатно) — пошаговая инструкция

## Что понадобится
- Аккаунт GitHub (бесплатно): https://github.com/signup
- Аккаунт Railway (бесплатно): https://railway.app
- Токен бота от @BotFather

---

## Шаг 1 — Создайте репозиторий на GitHub

1. Зайдите на https://github.com/new
2. Назовите репозиторий, например `visa-bot`
3. Поставьте галочку **«Add a README file»**
4. Нажмите **«Create repository»**

## Шаг 2 — Загрузите файлы бота

1. В созданном репозитории нажмите **«Add file» → «Upload files»**
2. Перетащите туда 3 файла:
   - `visa_bot.py`
   - `requirements.txt`
   - `Procfile`
3. Нажмите **«Commit changes»**

## Шаг 3 — Откройте таблицу для чтения

1. Откройте вашу Google Sheets таблицу
2. Нажмите **«Поделиться»** (вверху справа)
3. В разделе «Общий доступ» выберите:
   **«Все, у кого есть ссылка»** → **«Читатель»**
4. Нажмите «Готово»

## Шаг 4 — Разверните на Railway

1. Зайдите на https://railway.app и войдите через GitHub
2. Нажмите **«New Project»**
3. Выберите **«Deploy from GitHub repo»**
4. Найдите и выберите ваш репозиторий `visa-bot`
5. Railway начнёт деплой — **подождите, пока он не завершится**

## Шаг 5 — Добавьте токен бота

1. В Railway кликните на ваш сервис (прямоугольник с названием)
2. Перейдите во вкладку **«Variables»**
3. Нажмите **«+ New Variable»**
4. Добавьте:
   - **Name:** `BOT_TOKEN`
   - **Value:** ваш токен от BotFather (длинная строка вида `7123456789:AAH...`)
5. Railway автоматически перезапустит сервис

## Шаг 6 — Проверьте!

Откройте @VISATOUR_AB_bot в Telegram и отправьте `/start`.
Бот должен ответить списком команд.

---

## Важно

- **Railway даёт $5 бесплатных кредитов в месяц** — для бота этого хватает с запасом
- Бот работает 24/7, пока есть кредиты
- Если таблица закрыта (нет доступа по ссылке), бот выдаст ошибку при /refresh
- Данные из таблицы кэшируются на 5 минут, чтобы не перегружать Google
